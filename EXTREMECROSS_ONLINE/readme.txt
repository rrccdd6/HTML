Página desenvolvida para um box de Crossfit localizado em Mesquita-RJ.

A idéia do projeto é criar uma página que ajude a divulgar as promoções do estabelecimento e ter utilidades que tragam o cliente do box a usar de forma recorrente a página por conta das calculadoras. 

Ter um novo canal de contato com o cliente e com os links com o WhatsApp e Instagram. O contato por email e a localização facilitada por meio do mapa. 

Projeto feito em: HTML, CSS e JavaScript.
